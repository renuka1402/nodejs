# Personal Finance Tracker

An EJS + Express + MongoDB finance tracker with:

- user registration and login
- dashboard totals, budget progress, charts, and alerts
- profile management with profile picture upload
- category management
- income and expense tracking
- transaction history with search, filter, sort, pagination, CSV/PDF export
- budget creation and tracking
- financial reports with PDF export
- payment reminders

## Run

1. Copy `.env.example` to `.env`
2. Set `MONGO_URI` and `JWT_SECRET`
3. Run `npm install`
4. Run `npm run dev` or `npm start`

Open `http://localhost:3000`
